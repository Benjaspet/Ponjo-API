Endpoints

Welcome to the endpoint page of the Crescent API! Here, you can find all available endpoints, how to access them, and view sample responses. With over 50 endpoints, it's almost certain you'll be able to find this API of great use. Responses are easily-parsable in a straightforward JSON format to allow for seamless implementation into your application.

NOTE: All POST requests that require a URL to an image (for example, all image manipulation routes) require> the URI components to be properly encoded. Failure to do so will result in the request not going through. If you're not familiar with URI encoding, you can learn more here.

All Endpoints

Information & Statistics

GET REQUEST
/v1/status
Get the status of the API.

GET REQUEST
/v1/stats?endpoint={endpoint}
View statistics of a particular endpoint.

GET REQUEST
/v1/info
View information about the API.

Artificial Intelligence

GET REQUEST
/v1/chatbot?message={message}
Obtain a chatbot response message.

Game Queries

GET REQUEST
/v1/query/mcbe?host={host}&port={port}
Get the status of a Minecraft: Bedrock Edition server.

GET REQUEST
/v1/query/minecraft?host={host}&port={port}
Get the status of a Minecraft: Java Edition server.

Image Manipulation

POST REQUEST
/v1/images/trigger?image={image-URL}
Give an image a "triggered" GIF overlay.

POST REQUEST
/v1/images/gay?image={image-URL}
Make your image a little more gay!

POST REQUEST
/v1/images/jail?image={image-URL}
Give yourself a nice mugshot.

POST REQUEST
/v1/images/wanted?image={image-URL}
Make yourself wanted by the police.
