Deck Endpoint

One of the Ponjo API's most popular endpoints is the deck endpoint. It's a seamless virtual card deck simulator with the ability to create decks of cards, shuffle them, and even save them for later. Including built-in methods to evaluate poker hands, blackjack hands, and more, you'll be able to create functional card games with our API.

GET REQUEST
/v1/decks/create
Create a virtual deck of cards.

GET REQUEST
/v1/decks/find?id={deck-id}
Find a virtual deck by ID.

GET REQUEST
/v1/decks/shuffle?id={deck-id}
Shuffle a virtual deck by ID.

GET REQUEST
/v1/decks/poker/evalhand?hand={hand}
Shuffle a virtual deck by ID.

Image Manipulation Endpoint

Easily provide images and add numerous filters, effects, and flairs using our API. What's more, no API key is required for these powerful endpoints. Create awesome commands for your Discord bot with our easy-to-use image manipulation endpoints.

POST REQUEST
/v1/images/trigger?image={image-URL}
Give an image a "triggered" GIF overlay.

POST REQUEST
/v1/images/gay?image={image-URL}
Make your image a little more gay!

POST REQUEST
/v1/images/jail?image={image-URL}
Give yourself a nice mugshot.

POST REQUEST
